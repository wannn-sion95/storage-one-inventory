# 📦 StorageOne - Modern Inventory Management System

![StorageOne Banner](https://capsule-render.vercel.app/api?type=waving&color=0:0F2027,50:203A43,100:2C5364&height=220&section=header&text=StorageOne&fontSize=55&fontColor=ffffff&animation=fadeIn&fontAlignY=38&desc=Inventory%20Management%20System&descAlignY=60&descSize=18)
**StorageOne** adalah sistem manajemen inventaris (Gudang) tingkat *Enterprise* yang dibangun menggunakan Laravel 11. Aplikasi ini dirancang dengan antarmuka *Clean Minimalist* untuk memudahkan admin gudang dalam memantau, menambah, dan mengelola arus barang secara *real-time*.

## ✨ Fitur Utama

- **🔐 Autentikasi Aman:** Sistem Login dan Registrasi menggunakan Laravel Breeze.
- **📊 Dashboard Interaktif:** Ringkasan statistik ragam produk, total unit, dan peringatan stok kritis.
- **📦 Manajemen Produk (CRUD):** Tambah, baca, perbarui, dan hapus data barang dengan mudah.
- **🏷️ Relasi Kategori:** Pengelompokan barang berdasarkan kategori yang terstruktur (Relasi Database).
- **🖼️ Manajemen Gambar:** Fitur *upload* foto produk dengan *preview* dinamis dan penghapusan *file* fisik otomatis.
- **🔍 Pencarian & Paginasi (Server-Side):** Pencarian data yang cepat dan akurat tanpa membebani *browser*.
- **🎨 UI/UX Enterprise:** Desain responsif, modern, dan modular menggunakan Bootstrap 5 & arsitektur *Component-Based*.

## 💻 Tech Stack

- **Framework:** Laravel 11 (PHP)
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, Bootstrap 5, Vanilla JavaScript
- **Icons:** FontAwesome 6
- **Authentication:** Laravel Breeze
